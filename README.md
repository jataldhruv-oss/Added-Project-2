## Web Application Vulnerability Assessment using OWASP ZAP

## Introduction

This project demonstrates web application security testing using OWASP ZAP (Zed Attack Proxy). The assessment was performed on the AltoroJ demo banking application, which is intentionally vulnerable for educational and testing purposes.

## Objective

* Identify web vulnerabilities
* Perform automated penetration testing
* Analyze security weaknesses
* Generate professional vulnerability reports

## Target Website

http://demo.testfire.net

## Tool Used

* OWASP ZAP

## Vulnerabilities Identified

1. SQL Injection
2. Cross Site Scripting (XSS)
3. Missing CSRF Protection
4. Missing Security Headers
5. Information Disclosure
6. Cookie Security Issues

## Key Findings

* SQL Injection vulnerability detected in login functionality
* Reflected XSS vulnerability detected in feedback form
* Multiple missing security headers observed
* Sensitive information disclosure identified

## Reports Generated

* HTML Scan Report
* PDF Vulnerability Report

## Conclusion

The scan successfully identified several common web application vulnerabilities. This project demonstrates the importance of secure coding practices and regular security testing.
